**ವೇದ ವಿಜ್ಞಾನ ಗುರುಕುಲಂ** ಒಂದು ಸಂಪೂರ್ಣ ಮುಕ್ತ-ಮೂಲ (Open Source) ವೆಬ್ ಅಪ್ಲಿಕೇಶನ್ ಆಗಿದೆ. ಇದು ಗುರುಕುಲ ಪದ್ಧತಿಯಲ್ಲಿ ವಿದ್ಯಾರ್ಥಿಗಳನ್ನು **ಗಣಗಳ** (ತಪಃ, ತೇಜಃ, ಓಜಃ, ವರ್ಚಃ, ಪ್ರೇಯಃ, ಶ್ರೇಯಃ, ಭ್ರಾಜಃ, ಯಶಃ) ಆಧಾರದಲ್ಲಿ ವಿಂಗಡಿಸಿ, ಅವರ ದೈನಂದಿನ ಹಾಜರಾತಿ, ಪಾಠದ ದಾಖಲೀಕರಣ, ಪ್ರತಿ **ಪಕ್ಷ** (15 ದಿನಗಳಿಗೊಮ್ಮೆ) ವರದಿ ಕಾರ್ಡ್, ಮತ್ತು ಗಣವಾರು ಸಮಯ ವೇಳಾಪಟ್ಟಿಯನ್ನು ನಿರ್ವಹಿಸಲು ಅನುವು ಮಾಡಿಕೊಡುತ್ತದೆ.

## 📥 ಸ್ಥಾಪನೆ ಮತ್ತು ಬಳಕೆ

### 1. ನೇರವಾಗಿ ಬ್ರೌಸರ್‌ನಲ್ಲಿ ಬಳಸಲು
- ಈ repository ವನ್ನು clone ಮಾಡಿ ಅಥವಾ `index.html` ಫೈಲ್ ಅನ್ನು ಡೌನ್ಲೋಡ್ ಮಾಡಿ.
- ಯಾವುದೇ ವೆಬ್ ಬ್ರೌಸರ್ (Chrome, Firefox, Edge) ನಲ್ಲಿ `index.html` ಅನ್ನು ತೆರೆಯಿರಿ.
- **ಯಾವುದೇ ಸರ್ವರ್ ಅಗತ್ಯವಿಲ್ಲ** – ಎಲ್ಲಾ ಡೇಟಾ ನಿಮ್ಮ ಬ್ರೌಸರ್‌ನ localStorage ನಲ್ಲಿ ಉಳಿಯುತ್ತದೆ.

### 2. GitHub Pages ಮೂಲಕ ಆನ್‌ಲೈನ್ ನಲ್ಲಿ ಹೋಸ್ಟ್ ಮಾಡಲು
1. ಈ repository ಅನ್ನು ನಿಮ್ಮ GitHub ಖಾತೆಗೆ ಅಪ್ಲೋಡ್ ಮಾಡಿ.
2. Repository → `Settings` → `Pages` ಗೆ ಹೋಗಿ.
3. `Branch` ಅನ್ನು `main` (ಅಥವಾ `master`) ಎಂದು ಆಯ್ಕೆ ಮಾಡಿ ಮತ್ತು `Save` ಕ್ಲಿಕ್ ಮಾಡಿ.
4. ಕೆಲವು ನಿಮಿಷಗಳಲ್ಲಿ ನಿಮ್ಮ ಅಪ್ಲಿಕೇಶನ್ `https://<username>.github.io/<repo-name>/` ನಲ್ಲಿ ಲಭ್ಯವಾಗುತ್ತದೆ.

## 🗂️ ಡೇಟಾ ಬ್ಯಾಕಪ್ ಮತ್ತು ಮರುಸ್ಥಾಪನೆ

ಅಪ್ಲಿಕೇಶನ್ನ **ಕೊನೆಯ ಟ್ಯಾಬ್** ಆಗಿ “📀 Data Backup & Restore” ಸೇರಿಸಲಾಗಿದೆ.

- **Download Backup**: ಈ ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿದಾಗ, ಎಲ್ಲಾ ಡೇಟಾ (`students`, `teachers`, `subjects`, `dailyRecords`, `timetables`) ಒಂದು `.json` ಫೈಲ್ ಆಗಿ ನಿಮ್ಮ ಕಂಪ್ಯೂಟರ್‌ಗೆ ಡೌನ್ಲೋಡ್ ಆಗುತ್ತದೆ.
- **Restore from Backup**: ಹಿಂದೆ ಡೌನ್ಲೋಡ್ ಮಾಡಿದ `.json` ಫೈಲ್ ಅನ್ನು ಆಯ್ಕೆ ಮಾಡಿ “Restore” ಕ್ಲಿಕ್ ಮಾಡಿದರೆ, ಎಲ್ಲಾ ಡೇಟಾ ಪುನಃ ಸ್ಥಾಪಿತವಾಗುತ್ತದೆ (ಪುಟ ರೀಫ್ರೆಶ್ ಆಗುತ್ತದೆ).

> ⚠️ **ಸೂಚನೆ:** Restore ಮಾಡುವ ಮೊದಲು, ಪ್ರಸ್ತುತ ಡೇಟಾವನ್ನು Backup ಮಾಡಿಕೊಳ್ಳಿ. Restore ಮಾಡಿದ ನಂತರ ಹಳೆಯ ಡೇಟಾ ಮೇಲಕ್ಕೆ ಬರೆಯಲ್ಪಡುತ್ತದೆ.
